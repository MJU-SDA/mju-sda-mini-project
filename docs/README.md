# mju-sda-mini-project
명지대학교 통계적 데이터분석 미니 프로젝트 레포지터리

## 디렉토리
https://github.com/mju-sda-mini-project
```
├── data/                  # 수집한 데이터 저장 디렉토리
│   ├── 1.raw/             # 원본 데이터
│   ├── 2.processed/       # 전처리 완료된 데이터
│   ├── 3.output/          # 분석 결과 및 회귀 결과
|   |── 4.etc/             # 사용여부가 정해지지 않은 데이터
│
├── src/                   # 코드 디렉토리
│   ├── 1.preprocessing/   # 전처리 관련 코드
│   ├── 2.eda/             # 데이터 탐색+분석 관련 코드
│   ├── 3.regressioin/     # 회귀분석 관련 코드
│
├── docs/                # 문서 디렉토리
|   ├── README.md            # 프로젝트 설명 파일
```
