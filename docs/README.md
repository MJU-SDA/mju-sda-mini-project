## /data/processed/processed_final.csv

아래는 processed_final.csv에 포함된 주요 컬럼들의 정의입니다.
모든 데이터는 행정구역(시) / 행정구역(시군구) 단위로 구성되어 있습니다.

### 1. 행정구역 정보

시점: 데이터가 수집된 기준 연도

행정구역(시): 광역시·도 단위 지역명 (예: 서울특별시, 경기도 등)
행정구역(시군구)별: 시군구 단위 지역명 (예: 종로구, 서대문구, 제주시 등)

### 2. 인구 및 성별 구성

총인구수: 해당 시군구 전체 주민등록 인구
남자인구수: 해당 지역 전체 남성 인구(연령 구분 없음)
여자인구수: 해당 지역 전체 여성 인구(연령 구분 없음)

참고: 성별 데이터는 연령대와 함께 제공하기가 쉽지않아, 따로따로 진행

### 3. 연령대 인구 구성

아래 연령대는 남녀 전체를 합산한 인구 수를 의미하며, 모두 시군구 기준입니다.

영아기: 0~4세
아동기: 5~9세
청소년기: 10~19세
청년기: 20~39세
중년기: 40~59세
노년기: 60세 이상

### 4. 생활·환경 관련 지표

CCTV: 해당 시군구 설치된 CCTV 대수
1인가구비율: 전체 가구 중 1인가구 비율
경찰서수: 해당 시군구 내 경찰서 개수

### 5. 범죄 발생 지표

범죄 데이터는 경찰청의 세부 중분류 범죄 데이터를 기반으로 하며,
유사 범죄끼리 의미별로 묶어서 대분류 형태로 재구성한 값입니다.
(예: 강간·추행·미수 등을 하나로 묶어 “강력범죄\_성범죄”로 통합)
각 컬럼은 연간 발생 건수를 인구 수로 나눈 후 1000을 곱한 값으로, 1000건당 비율을 의미합니다.

강력범죄*살인계: 살인기수, 살인미수 등
강력범죄*강도: 강도 관련 범죄
강력범죄*성범죄: 강간, 유사강간, 강제추행 등
절도범죄*전체: 모든 절도 범죄를 합산
폭력범죄*전체: 폭행·손괴 등 폭력 관련 범죄
지능범죄*재산범죄: 사기, 횡령, 배임 등의 재산 관련 지능범죄

풍속범죄*전체: 성풍속, 도박 관련 범죄
특별경제범죄*전체: 특별경제범죄 전체
마약범죄*전체: 마약 관련 범죄
보건범죄*전체: 보건·위생 관련 범죄
환경범죄*전체: 환경 관련 법규 위반
교통범죄*전체: 교통 관련 법규 위반
노동범죄*전체: 노동 관련 위반
안보범죄*전체: 국가보안법 및 유사 범죄
선거범죄*전체: 선거 관련 범죄
병역범죄*전체: 병역법 위반
기타범죄\_전체: 위 항목에 포함되지 않는 기타 범죄

### 6. 데이터 전처리 관련 참고 사항

범죄 컬럼들은 원본 세부 항목들을 그룹화하여 합산한 값임
NaN 값은 같은 행정구역(시)(광역단위) 기준 평균값으로 채워서 데이터 왜곡을 최소화함

최종 merged_df에는
인구·연령대·성별·CCTV·1인가구비율·경찰서수·범죄지표들이 모두 포함되어 있어
바로 EDA 및 회귀 분석에 활용할 수 있음

## /data/2.processed/processed_for_analysis.csv

아래는 통계적 가설 검정 및 회귀분석을 위해 processed_final.csv를 기반으로 2차 가공(파생변수 생성 및 이상치 제거)을 마친 최종 데이터셋입니다.

### 1. 기본 컬럼 구성 (상속)

기존 processed_final.csv에 포함된 행정구역, 인구수(총/남/여/연령대), CCTV, 경찰서수, 1인가구비율, 세부 범죄율 컬럼을 모두 포함하고 있습니다.

### 2. 추가된 파생변수 (Feature Engineering)

분석의 정확도를 높이기 위해 절대 수치(인구수)를 **비율(%)**로 변환하고, 전체적인 치안 수준을 파악하기 위한 통합 지표를 생성했습니다.

인구 구조 비율 지표 (단위: %)

청년기\_비율: 전체 인구 대비 청년층(20~39세) 비율

노년기\_비율: 전체 인구 대비 노년층(60세 이상) 비율

남성\_비율: 전체 인구 대비 남성 비율

여성\_비율: 전체 인구 대비 여성 비율

통합 범죄 지표 (단위: 인구 1000명당 건수)

총범죄율: 개별 범죄 유형(절도, 폭력, 지능, 강력 등)의 발생 비율을 모두 합산한 지역별 종합 범죄 지수

### 3. 데이터 정제 및 전처리 내역 (Data Cleaning)

회귀분석 및 상관분석 시 통계적 왜곡을 방지하기 위해 다음과 같은 정제 과정을 거쳤습니다.

광역 단위 합계(이상치) 제거:

원본 데이터에 포함된 '서울특별시-서울특별시', '경기도-경기도'와 같은 광역자치단체 합계 행을 제거했습니다.

이유: 합계 데이터의 거대한 수치가 기초자치단체(시군구) 간의 실제 상관관계를 왜곡하는 현상(Outlier effect)을 방지하기 위함입니다.

### 분석 활용 가이드:

이 데이터셋은 이상치가 제거되고 비율 변수가 추가되어 있어, 별도의 가공 없이 바로 Spearman 상관분석, T-Test/Mann-Whitney U Test, 다중회귀분석의 Input 데이터로 사용할 수 있습니다.
